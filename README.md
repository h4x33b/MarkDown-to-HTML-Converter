## Installation

To run the tool locally, follow these steps:

1. **Clone this repository** to your local machine:
   ```bash
   git clone https://github.com/h4x33b/MarkDown-to-HTML-Converter.git
   ```

2. **Navigate to the project directory**:
   ```bash
   cd MarkDown-to-HTML-Converter
   ```

3. **Open `index.html`** in your browser:
   - Double-click on the `index.html` file, or use any local server setup you prefer (e.g., VS Code Live Server extension).

## Usage

1. **Markdown to HTML Conversion**:
   - Paste or type your Markdown text into the "Enter Markdown Text" textarea.
   - Click the **Convert to HTML** button to see the converted HTML output in the "Converted HTML Output" section.

2. **Download HTML**:
   - After converting your Markdown, click the **Download HTML** button to save the generated HTML as a `.html` file.

3. **Generate SEO-Friendly URL Slug**:
   - Enter a page title or keyword into the "Generate SEO URL Slug" input field.
   - Click **Generate Slug** to create a clean, SEO-optimized URL slug.

4. **Markdown Cheatsheet**:
   - A reference guide for common Markdown syntax is available on the page, including how to create headings, lists, bold/italic text, and more.

## Technologies Used
